# Video Stats Globals

Manage `client.video.stats.globals` endpoints.

::: details Why does this page use `globals_`?
The SDK resource module is named `globals_.py` to avoid a Python keyword collision,
while the public SDK accessor remains `client.video.stats.globals`.
:::
