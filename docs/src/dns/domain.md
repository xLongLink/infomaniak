# Domain

The `dns.domain` resource provides methods to list domains and inspect one domain.

## List Domains

- params
  - None
- returns
  - `None` (currently not implemented in the SDK)

```py
from infomaniak import Client

client = Client(token="<token>")
client.dns.domain.list()
```

## Display Domain

- params
  - None
- returns
  - `None` (currently not implemented in the SDK)

```py
from infomaniak import Client

client = Client(token="<token>")
client.dns.domain.display()
```
