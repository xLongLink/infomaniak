# Cloud Database Config

The `cloud.database.config` resource manages configuration entries for a managed cloud database.

## Methods

- `cloud.database.config.get(...)`
- `cloud.database.config.list(...)`
- `cloud.database.config.create(...)`
- `cloud.database.config.update(...)`
- `cloud.database.config.remove(...)`

::: warning
`cloud.database.config` endpoints are declared in the SDK but not implemented yet.
:::
